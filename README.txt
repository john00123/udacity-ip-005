Memory Game.


Objective: The games objective is to find all card pairs in the board.

How does it work: You select two cards, if they are a match they will remain turned, if not then they will flip upside down until you have found a pair.

This is a game of memory. Beat it in with the least number of moves possible to get 3 stars.

--------

Developer notes:

路 The code relies on Jquery, so it's recommended to enable JavaScript in your browser to enable full functionality.

路 Make sure you are running the latest version of Firefox, Chrome or Safari as there is a flex box dependency.

路 This game uses google fonts to render "Open sans".

路 The logic and structure of this game is dynamic. It will generate the number of cards and steps needed to win as necessary, you will need to make sure you add, two values on the initial card variable to avoid leaving the user with a odd number of cards.
